## Features

- **Modern Design**: Clean, responsive layout that looks great on all devices
- **10 Color Palettes**: Choose from 10 pre-designed color schemes
- **Dark Mode**: Full dark mode support for all color palettes
- **Search**: Fast, client-side search powered by Lunr.js
- **Tags**: Organize content with a comprehensive tagging system
- **Code Highlighting**: Beautiful syntax highlighting with copy functionality
- **GitHub Pages Ready**: Deploy directly to GitHub Pages
